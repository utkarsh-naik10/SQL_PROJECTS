# Revenue Dashboard Queries — Expert Level Analytics

This folder contains SQL queries used to build the **Revenue Dashboard** for the Chinook database. It includes:

* Five SQL files (one per visual)
* A `Result_Snapshots` folder containing output screenshots used for dashboard insights
